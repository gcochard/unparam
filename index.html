Unparam

This node module allows you to call out unused parameters explicitly. It is great for self-documenting code, and it silences jslint's unparam warning.

Usage:

var unused = require('unparam');
...
function(arg1,arg2){
    unused(arg1);
    do_something_with(arg2);
}
